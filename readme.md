# Tree Visualizer (BST & AVL)

Interactive Binary Search Tree and AVL Tree visualizer built using:

- HTML
- CSS
- Vanilla JavaScript
- SVG rendering

## Features

- BST insertion & visualization
- AVL self-balancing rotations
- Interactive UI
- Mode toggle between BST and AVL
- Automatic node positioning
